# Team Directory

This folder contains team-specific configurations and resources.

## Structure

```
team/
├── README.md          # This file
├── contacts.md        # Team contacts and escalation paths (optional)
├── workflows.md       # Team-specific workflows (optional)
└── ...                # Other team resources
```

## Usage

Add any team-specific documentation here that doesn't fit in the main workspace files.

Examples:
- Contact lists and escalation paths
- Team-specific workflows and processes
- Integration guides
- Runbooks
